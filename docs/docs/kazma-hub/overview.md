# Kazma Hub Overview

Kazma Hub is the central marketplace for discovering, publishing, and managing AI agent skills.

## What is Kazma Hub?

Kazma Hub is a SQLite-backed skill registry that enables:

- **Publishing** — Share your skills with the community
- **Discovery** — Search for skills by capability, tag, or author
- **Installation** — One-command skill installation
- **Validation** — Automated security and quality checks
- **Certification** — Quality assurance through certification levels

## Quick commands

```bash
# Search for skills
kazma hub search "drone"

# Install a skill
kazma hub install author/skill-name

# List installed skills
kazma hub list

# View skill details
kazma hub info author/skill-name
```
